# NOTE: before run local server

### Install node_modules

`npm i`

### Create .env.local and .env.production(depend purpose) with these following info with username=username & password=password

```
DB_URL=mongodb+srv://username:password@and_other_part_of_the_link
PORT=5000
DB_USERNAME=mongo_link_usrname
DB_PASSWORD=mongo_link_pass
```

## Run dev env

`npm run dev`
